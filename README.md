简介
----

Clash 是一个使用 Go 语言编写，基于规则的跨平台代理软件核心程序。  
ClashA 是目前在 Andorid 上表现较为出色的图形化 Clash 分支。  
软件下载[地址](https://github.com/ccg2018/ClashA/releases)  

**特点**  
- 可随时切换代理模式及节点  
- 支持节点批量延迟测试  
- 通过托管链接一键配置  
- 规则命中分析  
- 日志输出
